KEY ELEMENTS TO SHOW IN FIGMA:
==============================
1. Flow from top to bottom
2. Each box is a distinct step
3. Decision diamond for "Review Plan"
   - YES path (green) continues forward
   - NO path (red) loops back to "Fix code"
4. Color coding:
   - Steps 1-3: Blue (planning phase)
   - Decision: Yellow (review)
   - Apply: Green (execution)
   - Success: Green with checkmark
   - Verify: Purple (testing)
   - Destroy: Red (cleanup)
5. Show approximate time for each step:
   - init: ~30 seconds
   - plan: ~10 seconds
   - apply: 2-3 minutes
   - verify: ~1 minute
6. Add icons:
   - Download icon for init
   - Magnifying glass for plan
   - Gear/settings for apply
   - Checkmark for success
   - Test tube for verify
   - Trash can for destroy
